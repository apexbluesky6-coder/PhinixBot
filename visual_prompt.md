# Visual Prompt: PhinixRemo Dashboard Mockup

This prompt is designed for image generation models (Midjourney, DALL-E 3, Stable Diffusion) to visualize the Version 1 PhinixRemo dashboard.

## Image Generator Prompt

> **Prompt**: A wide dashboard screen in a dark-mode interface, showing several panels. 
> 
> **Left Side**: A column with logos and generic icons representing multiple crowdsourcing platforms (Toloka, Appen, Clickworker).
> 
> **Center**: A task list table with columns: "Platform", "Task Type", "Pay", "Time", "AI-assist (colored indicator)", and priority tags (High/Low).
> 
> **Right Side**: A floating side panel titled "PhinixRemo Co-pilot" explaining a task in simple language with bullet points and a warning icon for "Attention Checks".
> 
> **Background Elements**: Several stylized human silhouettes at different desks, each with their own smaller screen, connected to the central dashboard with glowing lines, representing multiple workers.
> 
> **Aesthetic**: Efficient, focused, high-tech, "Human-in-the-Loop" aesthetic, vibrant blue and charcoal colors, clean glassmorphism UI elements, ultra-detailed 4k render.

## UI Components to Include in Development
- **Status Pills**: `NEW`, `IN PROGRESS`, `REVIEW`, `SUBMITTED`.
- **EHR Ticker**: "Effective Hourly Rate: $12.50/hr".
- **Trap Alerts**: High-contrast warning messages from the AI brain.
- **Worker Switching**: A profile switcher to view sub-worker progress.
