# Toloka Foundation Guidelines

## 1. Quality Policy
Toloka employs "Honey Pots" (Attention Checks). These are tasks with known answers. 
- **Rule**: If accuracy drops below 80%, access to the project is revoked.
- **Phinix Tip**: Look for very obvious questions mixed with hard ones.

## 2. Speed Limits
Never submit a task faster than 5 seconds. This flags "Bot Behavior".
- **Bot Guardrail**: PhinixRemo will add a 7-second countdown before enabling the "Submit" button on the UI.

## 3. Ban Prevention
Avoid using VPNs. Toloka tracks IP consistency.
