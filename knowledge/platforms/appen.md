# Appen Connect (China & Global) Guidelines

## 1. Multi-Account Restriction
Appen is extremely strict about IP and device matching. 
- **Rule**: Never use the same device for multiple accounts even if on different IPs.
- **Phinix Tip**: We recommend dedicated virtual machines or hardware for each worker.

## 2. Invoicing Accuracy
Pay is usually monthly. Ensure all hours are logged in the "Earnings" tab correctly.
- **Phinix Logic**: The bot will auto-calculate estimated earnings per project to cross-reference with Appen's dashboard.

## 3. Qualification Tests
Many Appen projects require a 100+ page PDF read.
- **Phinix Support**: Upload the PDF to the Knowledge Base. The "Explainer Agent" will provide a searchable RAG index for instant rules lookup.
